# email-messenger
Простой децентрализованный мессенджер на основе электронной почты. Позволяет обмениваться сообщениями и файлами через обычные email-аккаунты, используя существующую email-инфраструктуру.

## Особенности

- **Децентрализованность** - не требует центрального сервера
- **Групповые чаты** - создавайте группы и общайтесь с несколькими участниками
- **Поддержка вложений** - отправляйте и получайте файлы
- **Реальное время** - мгновенное получение сообщений через IMAP IDLE
- **История сообщений** - все сообщения сохраняются локально
- **Счетчик непрочитанных** - визуальное отображение новых сообщений
- **Псевдонимы контактов** - удобные имена для email-адресов
- **Управление участниками** - добавляйте/удаляйте участников групп

## Требования

- Python 3.8 или выше
- PyQt6
- Доступ к email-аккаунту с поддержкой SMTP и IMAP

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/anton-trnt/email-messenger.git
cd email-messenger
```

2. Сборка:
У вас должен быть установлен питон, далее нужно установить PyQt6:
```bash
pip install PyQt6
```

Если нужен .exe файл, устновите PyInstaller:
```bash
pip install PyInstaller
```

После этого под Windows можно собрать исполняемый файл:
```bash
python -m PyInstaller msg.spec
```

Либо просто запускайте скрипт:
```bash
python msg.py
```

## Использование
1. Основные функции:

1.1 Добавление контакта

- Нажмите кнопку "➕ Добавить контакт"

- Введите псевдоним и email-адрес

1.2 Создание группы

- Нажмите кнопку "👥 Добавить группу"

- Введите название группы и список участников через запятую

1.3 Отправка сообщения

- Выберите контакт или группу из списка слева

- Введите текст в поле ввода

- Нажмите Enter или кнопку "Отправить"

1.4 Отправка файлов

- Выберите чат

- Нажмите кнопку "📎 Вложение"

- Выберите файлы для отправки

- Введите текст сообщения (опционально)

- Нажмите "Отправить"

2. Управление группой

- Нажмите правой кнопкой мыши на группе

- Выберите "Участники группы"

- Изменяйте псевдонимы участников

- Добавляйте или удаляйте участников

- Двойной клик по email участника откроет чат с ним

3. Настройка псевдонимов

- Для контакта: правый клик → "Изменить псевдоним"

- Для участника группы: измените в окне управления группой

4. Горячие клавиши
   
- Enter - отправить сообщение

- Shift+Enter - новая строка в сообщении

## Структура данных

Приложение создает следующие файлы и директории:

```text
data/
├── contacts.json           # Список контактов и групп
├── processed_uids.json     # UID обработанных сообщений
├── last_scan_time.txt      # Время последнего сканирования
├── [email]/                # Папка контакта
│   ├── log.txt            # История переписки
│   ├── last_time.txt      # Время последнего сообщения
│   └── attachments/       # Полученные вложения
└── group_[hash]/           # Папка группы
    ├── log.txt
    ├── last_time.txt
    ├── attachments/
    └── group_[hash]_aliases.json  # Псевдонимы участников
```

## Ограничения
Все участники должны использовать одинаковый префикс темы сообщения

Размер вложений ограничен возможностями почтового сервера

Требуется стабильное интернет-соединение для работы в реальном времени

## Устранение неполадок
1. Сообщения не приходят

- Проверьте правильность IMAP_FOLDER в конфигурации

- Убедитесь, что папка существует в почтовом ящике

2. Не удается отправить сообщение
   
- Проверьте SMTP настройки в конфигурации

- Убедитесь, что пароль почты действителен

- Проверьте, не блокирует ли почтовый сервис отправку
